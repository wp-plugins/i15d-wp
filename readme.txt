IDN Permalinks for WordPress.

== Description ==

Did you ever want to use more chars on your permalinks?
`i15d WP` is a really simplistic plugin that will let WordPress to be more
`soft` with chars restriction.

== Installation ==

1. Upload `i15d-wp.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
